Creature Evolution Arena V61 - Carte auto-fit + campi alti

Modifiche principali:
- Le carte nel campo si adattano meglio alla larghezza disponibile.
- Le informazioni interne delle carte si rimpiccioliscono in modo proporzionato.
- Nome, costo, badge, HP e ATK restano leggibili anche con carte più piccole.
- Le carte nel campo non vengono più tagliate verticalmente.
- Tuo campo e campo avversario sono leggermente più alti.
- Cache aggiornata a v61.

Per modificare rapidamente:
Apri style.css e cerca: V61 - Carte auto-fit

Valori principali:
--field-h: altezza esterna dei campi
--field-inner-h: altezza interna dove stanno le carte
--field-card-w: larghezza carte nel campo
--field-card-h: altezza carte nel campo
--hand-card-w: larghezza carte in mano
--hand-card-h: altezza carte in mano
